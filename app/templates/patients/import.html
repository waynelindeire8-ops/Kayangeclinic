Excel Format Guide
How to Create Your Excel File
  1. Open Microsoft Excel or Google Sheets
  2. Create a new spreadsheet
  3. Add column headers in the first row
  4. Add patient data starting from row 2
  5. Save as .xlsx format

Accepted Column Names

First Name: First Name, FirstName, Given Name, Fname

Last Name: Last Name, LastName, Surname, Family Name, Lname

DOB: DOB, Date of Birth, Birth Date, Birthday

Gender: Gender, Sex

Phone: Phone, Phone Number, Mobile, Cell, Telephone

Email: Email, Email Address

Address: Address, Home Address, Street Address

Emergency Contact: Emergency Contact, Next of Kin

Emergency Phone: Emergency Phone, Next of Kin Phone

Blood Group: Blood Group, Blood Type

Insurance: Insurance, Scheme Provider, Provider, Scheme

Scheme Type: Scheme Type, Plan, Insurance Type

Policy Number: Policy Number, Member Number, Insurance Number

Patient IDs are auto-generated (KMC-XXXX). Do not include an ID column.
